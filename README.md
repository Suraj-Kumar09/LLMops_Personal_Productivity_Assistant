# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/LLMops_Personal_Productivity_Assistant.git

cd LLMops_Personal_Productivity_Assistant
```

---

## 2️⃣ Create Virtual Environment

### Conda

```bash
conda create -n llmops python=3.12 -y

conda activate llmops
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Configure Environment Variables

Create a `.env` file and add your API keys.

```env
OPENAI_API_KEY=your_api_key
```

---

## 5️⃣ Run the Application

```bash
python app.py
```

The application will start locally and can be accessed from your browser.
